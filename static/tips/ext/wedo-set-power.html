Set the power of motors or lights to a specified value and turn them on. The value ranges from 0 to 100 (0 is no power and 100 is the maximum power).

Change the speed of the motor using the up and down arrow keys.


You can select a LEGO-WeDo-compatible device from the dropdown menu:
  • If you have only one motor, use the "motor" option, which will set the motor''s power regardless of where it is connected on the WeDo hub.
  • If you have two motors which you want to control separately, you can use "motor A" with one and "motor B" with the other.
  • "lights" sets the lights'' power regardless of where they are connected on the WeDo hub.
  • "everything" sets the power for all the connected motors and lights.

To use set power, you need WeDo motor(s) or lights connected to your computer via a LEGO WeDo hub.

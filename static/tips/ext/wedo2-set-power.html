Set the power of a motor to a specified value and turn it on.

Example

Change the speed of the motor using the up and down arrow keys.

Notes

The value of set power ranges from 0 to 100 (0 is no power and 100 is the maximum power).

You can select a LEGO WeDo motor from the dropdown menu. Select 'motor' to turn on any motors connected to the WeDo hub. If you have two motors which you want to control separately, use 'motor A' for one and 'motor B' for the other.

Note: LEGO recommends only connecting one motor to the hub to limit power consumption.

To use set power, you need a LEGO WeDo2 motor connected to your computer via a LEGO WeDo2 hub.

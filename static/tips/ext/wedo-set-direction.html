Set the direction of the motor(s). This block does not turn them on if they are already off.

Run the motor for 10 seconds while switching the spinning direction every second.


You can select WeDo motor(s) from the first dropdown menu:
  • If you have only one motor, use the "motor" option, which will set the motor''s direction regardless of where it is connected on the WeDo hub.
  • If you have two motors which you want to control separately, you can use "motor A" with one and "motor B" with the other.
  • "all motors" sets the direction for all the connected motors.

You can select the direction from the second dropdown menu.

To use set direction, you need WeDo motor(s) connected to your computer via a LEGO WeDo hub.

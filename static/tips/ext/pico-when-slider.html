Run a script when the value of one of the PicoBoard's sensors becomes greater (or less) than a specified value.

Change the sprite's costume each time you clap.

You can select one of the seven PicoBoard sensors. Each reports a value in the range from 0 to 100.

The PicoBoard comes with four pairs of alligator clips that can be plugged into the inputs A-D. The value of the resistance sensor depends on how much electrical resistance there is between the metal ends of the alligator clips.


You can also select a greater or less than comparison from the second dropdown menu.

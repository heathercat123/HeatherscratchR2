LEGO WeDo 1.0 or PicoBoard Setup

Follow these steps to setup the Picoboard or the LEGO WeDo 1.0 kit.

(If you have the newer LEGO WeDo 2.0 set, see LEGO WeDo 2.0 Setup)

IMPORTANT: Install the Plug-In

You'll need to install a plugin to use Scratch with the LEGO WeDo 1.0 or Picoboard. Choose the plugin to download based on your computer and the web browser you use for Scratch. If you use a browser other than Chrome, choose one of these two options:

  • Windows (other browsers)
  • Mac (other browsers)

If you use the browser Chrome, you will need to download and install two pieces of software:

  • Windows (Chrome) and helper here
  • Mac (Chrome) and helper here

You will need administrator access on your computer to install the plugin.

Status: Hardware Devices

Red light means that the browser plugin has not been installed.

Yellow light means that the device is not connected or detected.

Green light means the device is ready to go!

Troubleshooting
If it's not working, try:
  • Make sure you have opened and installed the correct plug-in file for your computer and browser.
  • Make sure you have set permission in your browser preferences to allow the plug-in to run.
  • Restart your browser.
  • Make sure that you don't have Scratch open in another tab or browser.
  • Make sure the hardware is plugged in.

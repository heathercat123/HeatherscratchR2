Run a script when the tilt sensor changes position

When the tilt sensor gets tilted, the script will run.

The tilt sensor reports a number from 0 to 4. 0 means not tilted, 1 tilted down, 2 tilted right, 3 tilted up, and 4 tilted left.

Use the dropdown menu to choose = (equals) or not = (not equal to), then type in a number from 0 to 4.

To use when tilt, you need a tilt sensor connected to your computer via a LEGO WeDo hub.

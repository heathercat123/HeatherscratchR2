Report whether the button is pressed or one of the inputs A-D is connected.

The sprite will spin as long as the button is pressed.

To view the value of the sensor, click the checkbox next to the block in the palette.

You can select the button or one of the four inputs numbered A-D.

The PicoBoard comes with four pairs of alligator clips that can be plugged into the inputs A-D. An input is connected if there is a good electrical connection between the metal ends of the clips.

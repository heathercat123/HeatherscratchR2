Report the value detected by one of the PicoBoard sensors in the range from 0 to 100.

Make the sprite move up and down using the slider.

To view the value of the sensor, click the checkbox next to the block in the palette.

You can select one of the seven PicoBoard sensors. Each reports a value in the range from 0 to 100.

The PicoBoard comes with four pairs of alligator clips that can be plugged into the inputs A-D. A resistance sensor value block reports a number from 0 to 100, depending on how much electrical resistance there is between the metal ends of the alligator clips.
